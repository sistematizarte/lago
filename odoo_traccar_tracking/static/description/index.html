Odoo Fleet Traccar Tracking

"Track your vehicles with Traccar in Odoo Fleet!"

Odoo Fleet Traccar Tracking module facilitates you to track the vehicle movements of all your drivers via traccar mobile app. Moreover, the module facilitates you to manage tracking data and report it via Odoo.

Transportation is one of profitable businesses leading the market; however, you need to look into each part to run the business flawlessly. Whether you are running cab services or moving freight; the most important part is tracking the vehicles as you have the responsibility of the people in the cabs and goods in the freight.

Traccar is the best solution for tracking your fleet efficiently. It is open source and also lets you keep track of the details of your vehicle with zero possibilities of failure.

On the other hand, Odoo which is one of the topmost backend management software has offered a fleet module for the ease of the users. Odoo fleet lets the users to track as well as manage the data related to the vehicle(s).

Odoo Traccar Fleet Tracking Integration!

Integrating traccar app with Odoo fleet so that you can easily manage the tracking of your vehicles from a single software. It facilitates you to audit the movements of the vehicles; moreover, you can also check the travel history for a particular custom time period.

Prerequisites

  • Admin needs to download traccar web client as per his/her operating system; the link to download the app is
    https://www.traccar.org/download/.
  • Driver needs to install Traccar Client App for iOS and Android
    Link to install the app for iOS:
    https://apps.apple.com/us/app/traccar-client/id843156974#?platform=iphone
    Link to install the app for android:
    https://play.google.com/store/apps/details?id=org.traccar.client
  • Support Openstreet Map
  • (Required for google map only) You require Google Map API which you can create API key with the help of this link
    (https://console.cloud.google.com/apis/credentials)
    • After creating API key, you need to enable below mentioned APIs:
    • Geocoding API
    • Maps JavaScript API
    • Maps Static API
    Please go through the link mentioned here
    (https://developers.google.com/maps/documentation/directions/usage-and-billing)
    For the directions API Usage and billing.

Live Track your Vehicles from Odoo via Traccar

Odoo Fleet Traccar Tracking allows you to track the movement of the vehicles anytime. Hence, you know the current situation of your vehicle which means whether the vehicle is running or not.

Now, you don't need to worry about your vehicle as you can track each detail like movements in meters; even the battery percentage of the driver's cellphone.

Moreover, you can also check the accuracy of the vehicles; you can set the accuracy limit from your traccar application.

Check All your Online Vehicles Easily

Checking whether your drivers are online or offline can help you to know everything that is going on the rides. Hence, whenever a driver is on a ride then the status should be online.

The module helps you to differentiate the status of vehicles by its color; green color means that vehicle is online and red color means that vehicle is offline.

Moreover, you can also check whether the driver is moving from his/her location or not.

Key Features

Below is the detailed list of features for Odoo Fleet Traccar Tracking module

Integrate Traccar Mobile App to Odoo

  • Now, you integrate your Traccar iOS and Android app with Odoo.
  • Make successful connections via traccar credentials in Odoo end.

Associate vehicles with traccar device id

  • You can create as many vehicles you want in Odoo
  • Associate newly created or old one to the traccar device id

Track all Associated Vehicles from Odoo

  • You can live track the location of the vehicles individually
  • Or, you can also track the location of multiple vehicles together in a map

View the Trip history of any vehicle

  • You can check the tracking history of any vehicle for every 5 minute
  • Also, check the trip history of a particular vehicle on the basis of time duration

Map Integration

  • The module is fully integrated with google map and openstreet map
  • Hence, the map view used is google map or openstreet map

Check the Drivers' detailed information

  • You check the battery percentage of the driver's cellphone
  • Also, check the accuracy of the current location
  • You can also check the speed of the vehicle

Configure Live Tracking as you want

  • Live Tracking via scheduler in every 2 mins is available by default
  • Or, change the scheduler time as you require

Configure Tracking Feature for Drivers' Mobile Device

  • You can easily configure tracking feature in any iOS or android app
  • Enable/disable tracking feature for any particular device

Easily Import vehicle trip History

  • You can manually Import vehicle trip history for particular time duration

Check the total Trips of a Vehicle

  • You can check the count of the total trips of any vehicle
